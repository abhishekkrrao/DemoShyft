# Commands

## Rum commands

 For MacBook
> cd android && ./gradlew clean && cd .. && npm run android

For Windows
> cd android && gradlew clean && cd .. && npm run android

----

## Generating the APK

### for .apk file

> cd android && ./gradlew assembleRelease && cd ..

> cd android && ./gradlew clean && ./gradlew assembleRelease && cd ..

### for .aab file

> cd android && ./gradlew bundleRelease && cd ..
